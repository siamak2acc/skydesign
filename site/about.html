About

A static-first travel discovery site

SkyDesign Flight Deal Finder is built for fast static hosting today and secure affiliate or API integrations later.

Deployment approach

The site is authored as static HTML, CSS, and JavaScript, then built locally into clean URL folders and uploaded to Hostinger public_html over FTP. Cloudflare remains responsible for DNS.

When live pricing or partner APIs are added, secrets should stay in a backend helper, serverless function, or private API service. The static frontend should never contain API keys.
